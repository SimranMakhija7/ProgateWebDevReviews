About

This is an initiative under the Progate Mentorship Program to encourage mentees to begin contributing to Open Source and participate in HacktoberFest. On this platform, mentees and other Progate users can contribute to the repository and share their experiences with learning on Progate.
Progate is an online learning platform built to help beginners take their first steps in the world of programming, to take their first step towards becoming independent developers.
To start off, we would love to hear your experience with the NodeJS Web Development Path on Progate.

How can I contribute?

  1. Complete the NodeJS Web Development Path on Progate.
  2. Go to this Github repository.
  3. Use the provided issue template to create an issue on the repository.
  4. Create a new branch in the repository with the name of branch the same as your full name.
  5. Clone the repository.
  6. Navigate to your branch using the git checkout command in the command line.
  7. In your branch, open the index.html file and use the provided sample review card as template and add a card sharing your experience with the NodeJS learning path on Progate.
  8. Stage, commit and push your changes.
  9. On Github, go to your branch and create a Pull Request(PR) and link it to the issue you had created earlier.
  10. Submit your PR for review by any of the mentors.
  11. Your PR will be merged to master by one of the mentors and Voila! Your review is now on the site!
